# site-visitors

## Warning

## 部署
1. 新建一个 `命名空间` 名称 `site-visitors`
2. 新建一个 `Worker` 名称随意
3. 进入到刚才新建 `Worker` 的 `设置`，绑定KV命名空间  `NAMESPACE:site-visitors`
4. 脚本内容使用 [index.js](https://github.com/starxg/site-visitors/blob/main/index.js)
