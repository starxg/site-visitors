## site-visitors

### Deploy
[releases](https://github.com/starxg/site-visitors/releases)
```text
 -b <arg>   Save binlog to a <file>
 -h         usage help
 -l <arg>   Listen on address <addr> (default is 0.0.0.0)
 -p <arg>   Listen on TCP port <port> (default is 80)
```
